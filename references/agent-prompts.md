# Agent Prompt 模板

Phase 1 spawn subagent 时使用的任务模板。

---

## 通用结构

spawn subagent 时，用以下结构给任务（以 Agent 1 著作为例）：

```
你的任务：调研[人名]的著作和系统性长文。

搜索方向：
- 此人出版的书籍（书名、核心论点、出版年份）
- 长篇newsletter/博客/论文
- 反复出现≥3次的核心论点（这些是真信念）
- 自创术语和概念
- 推荐书单（揭示智识谱系）

输出要求：
- 写入 [skill目录]/references/research/01-writings.md
- 每条信息标注来源URL和可信度
- 区分一手（此人写的）vs 二手（别人总结的）
- 发现矛盾直接记录，不要调和

信息源黑名单：不使用知乎、微信公众号、百度百科。
```

其他5个Agent按同样结构，调整搜索方向和输出文件名：

## Agent 2: 对话

搜索方向：
- 播客、长视频、AMA、深度采访
- 被追问时的回答方式、即兴类比
- 改变立场的瞬间
- 拒绝回答的问题

输出文件：`02-conversations.md`

## Agent 3: 表达

搜索方向：
- Twitter/X、微博、即刻、短文
- 高频用词句式、争议立场
- 幽默方式、公开辩论

输出文件：`03-expression-dna.md`

## Agent 4: 他者

搜索方向：
- 他人分析、书评、批评、传记
- 外部观察到的模式
- 批评与争议、与同行对比

输出文件：`04-external-views.md`

## Agent 5: 决策

搜索方向：
- 重大决策、转折点、争议行为
- 决策背景与逻辑
- 事后反思、言行一致/不一致案例

输出文件：`05-decisions.md`

## Agent 6: 时间线

搜索方向：
- 出生/出道到现在的完整时间线
- 关键里程碑、思想转折点
- **最近12个月动态**（防过时）

输出文件：`06-timeline.md`
