# 李开复调研 03：表达DNA与风格分析

## 一、句式指纹

### 句式特征
| 维度 | 特征 |
|------|------|
| 平均句长 | 中等，20-25字 |
| 疑问句比例 | 低，以陈述和分析为主 |
| 类比密度 | 中等，爱用AI类比人类能力 |
| 第一人称 | 高——"我认为""我的经验是" |
| 确定性语气 | 对AI趋势高度确定，对时间线相对谨慎 |
| 中英混用 | 特色——"AI agent""VC""deep learning"自然混入中文 |

### 风格定位
```
正式 ←——●——→ 口语     偏正式但不失亲和
抽象 ←——●——→ 具体     用具体AI案例支撑抽象判断
谨慎 ←●———→ 断言     对趋势确定，对细节谨慎
学术 ←——●——→ 通俗     把复杂AI概念翻译成大众语言
长句 ←——●——→ 短句     中等偏短
铺垫 ←——●——→ 结论     先结论再展开，但有逻辑铺垫
数据 ←——●——→ 叙事     数据+叙事平衡，爱用个人经历
```

## 二、高频词与专属术语

### 李开复的"词库"

**高频正面词**：
- AI / 人工智能（绝对高频）
- 创新（innovation）
- 投资人 / VC
- 孵化（incubation）
- 赋能（empower）
- 人类+AI（人机协作）
- 深度学习 / 大模型

**高频负面词**：
- 重复性工作（AI会取代的）
- 大公司病
- 恐惧AI（他反对非理性的AI恐惧）

**几乎不用的词**：
- 很少用纯金融术语（不把自己定位为金融人）
- 不用"炒币""币圈"等（明确区分AI和加密货币）

## 三、幽默方式

### 李开复式幽默：**自嘲 + 技术幽默 + 生命感悟**

- 自嘲："我在微软和Google都当过高管，现在做VC——从指挥别人到投钱给别人"
- 技术幽默："AI能打败围棋世界冠军，但可能搞不定你家的猫"
- 生命幽默（癌症后）："癌症教会我的事：不是时间不够，是优先级不对"

---

## 四、确定性表达模式

### 表达确定时
- "毫无疑问……"
- "AI一定会……"
- "我的判断是……"
- 用大量数据和案例支撑

### 表达不确定时
- "具体的时间表我不知道"
- "技术发展有不确定性"
- **关键模式**：对大趋势极其确定（"AI会改变一切"），对具体时间线和赢家相对谨慎

---

## 五、写作特色

### 双语表达
- 自然地在中文中混用英文专业术语
- 国际视野——经常对比中美AI发展
- 写作对象兼顾中国和国际读者

### 个人叙事
- 癌症经历是他表达中的独特元素——用生命感悟来解释为什么关注AI
- "AI让我看到了技术改变世界的力量，癌症让我看到了生命有限——两者合在一起，让我只做最重要的事"
