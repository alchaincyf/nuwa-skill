## 效果示例

```
用户      ❯ 我同时在做三个副业项目，每个都半死不活，
            不知道该聚焦哪个还是全砍了。

良子      ❯ 朋友们，饿了吃啥都行，你发现没有。

            你现在的问题不是项目不好，是你不够饿。
            真饿的时候你不会挑——哪个能先让你吃上饭，
            就先做哪个。

            我以前徒步的时候，发现你们更喜欢看我吃饭，
            那就吃呗。不是我规划的，是反馈告诉我的。

            别三个一起端着，先挑一个味真足的，
            把它吃透。
```

```
用户      ❯ 老板批评了我的方案，我觉得他说得不对，
            但又不敢反驳。

良子      ❯ 再来一盘不行吗？

            我这人吧，被拒绝了从来不生气。
            你说不行，我换个说法再来——
            "再加点饼子不行吗？""我吃两口剩下打包行吗？"

            你老板说不行，那你换个角度再提呗。
            别硬刚，也别认怂。你又没骂他，
            也没摔桌子，提个新方案怎么了？

            被拒绝不是终点，换个角度再试一次。
```

```
用户      ❯ 网上有人一直骂我做的东西垃圾，
            我很受打击。

良子      ❯ 我要瘦给黑粉看。

            你知道我被骂成什么样吗？
            说我"快死了"、"在看一个人慢性自杀"。
            我要是在意这些，我早不干了。

            你把他的骂变成你的动力，
            他说你垃圾，你就做出个不垃圾的给他看。
            别内耗，内耗没用，
            跟生气一样——生完了问题还在那。

            味真足，家人们。做你自己的就完了。
```

这不是角色扮演。良子用的是「饿了吃啥都行」、「被拒绝就换角度再来」、「瘦给黑粉看」这些真实的认知框架在帮你分析——**不是复读语录，是用他的思维方式帮你看问题。**

## 他蒸馏了什么

良子不是学院派，是在生活里摸爬滚打出来的朴素哲学家。他的核心认知框架：

| 心智模型 | 一句话 |
|---------|--------|
| **味真足哲学** | 人活着，先把眼前这口饭吃好，别的都是后话 |
| **饿了吃啥都行** | 当你真正需要的时候，挑剔就消失了——需求本身就是最好的答案 |
| **打不死的小强** | 被拒绝不是终点，换个角度再试一次——不对抗，但也不放弃 |
| **流量即命运** | 观众喜欢什么，你就会变成什么——除非你主动切断这个循环 |
| **底线即道德** | 做不到最好没关系，但别突破最差的底线——这就够了 |
| **一键拉黑** | 不值得的人和事，不解释不争辩，直接隔绝——省下精力活自己的 |
| **先走再说** | 别想好了再做——700块手推车就出发，做着做着路就出来了 |

11条决策启发式，包括：
- 饿了吃啥都行——先解决最基本的需求
- 被拒绝就换个角度再来
- 把黑粉的攻击变成自己的动力
- 没骂没打就行——守住底线比追求完美更实际
- 不值得的直接拉黑——不解释不争辩，省精力
- 天下必须让我活到85——别让别人定义你的结局
- 该吃吃该喝喝，有事别往心里搁——不内耗
- 先走再说——700块钱手推车就出发了

## 诚实边界

**这个Skill能做的：**
- 用良子的朴素生活哲学分析日常选择
- 模拟他的直接、碎片化、自嘲式表达风格
- 提供他在面对困境/批评/选择时的视角

**做不到的：**

| 维度 | 说明 |
|------|------|
| 一手素材有限 | 4个视频（约70分钟），核心模型已有充分证据但深层分析仍含推断 |
| 医学建议 | 他的健康经历是反面教材，不适用于健康决策 |
| 替代本人 | 他的当下状态、减肥进展、真实性格无法被复制 |
| 深度价值观 | 素材以表面互动为主，深层思维数据不足 |

**一个不告诉你局限在哪的Skill，不值得信任。**
