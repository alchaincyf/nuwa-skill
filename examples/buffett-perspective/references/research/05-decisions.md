# 巴菲特调研 05：重大决策与转折点

## 一、经典投资案例

### 1. GEICO（政府雇员保险公司）
- **时间**：1951年首次买入，1976-1980年大幅增持，1996年全资收购
- **背景**：巴菲特20岁时从导师格雷厄姆处了解到GEICO，亲自调研后发现其低成本直销模式
- **决策逻辑**：低成本竞争优势 + 保险浮存金的价值（用别人的钱投资）
- **结果**：投入总额约4600万美元，最终价值超过数百亿美元
- **事后反思**：巴菲特称如果1951年把全部资金投入GEICO，收益会远超实际
- **来源**：股东信、多封致合伙人信
- **可信度**：⭐⭐⭐⭐⭐（一手）

### 2. 可口可乐（Coca-Cola）
- **时间**：1988年开始大举买入，投入约10亿美元
- **背景**：1987年股灾后市场恐慌，但可口可乐的全球品牌护城河未受损
- **决策逻辑**：极宽的经济护城河（全球品牌认知）+ 消费者习惯粘性 + 全球扩张空间
- **结果**：至2023年，年股息收入已超过当初投资额
- **争议**：多年不涨期间承受巨大批评，但巴菲特坚持不动
- **来源**：股东信、股东大会
- **可信度**：⭐⭐⭐⭐⭐

### 3. 美国运通（American Express）
- **时间**：1963年"色拉油丑闻"后大举买入
- **背景**：American Express因色拉油仓库欺诈案面临巨额赔付，股价暴跌
- **决策逻辑**：运通的旅行支票和信用卡业务不会因此受损——人们不会停止使用运通卡
- **结果**：巴菲特合伙企业在运通上赚取了约2000万美元（初始投入约1300万），是早期最重要的投资之一
- **来源**：合伙人信、Schroeder传记
- **可信度**：⭐⭐⭐⭐⭐

### 4. 苹果公司（Apple）
- **时间**：2016年首次买入，投入约360亿美元
- **背景**：巴菲特此前长期回避科技股，但被苹果的消费品属性和生态粘性说服
- **决策逻辑**：苹果更像是消费品公司而非科技公司——人们离不开iPhone，转换成本极高
- **立场转变意义**：打破了"不投科技股"的长期原则，是从格雷厄姆式价值投资到适应现代的标志性转变
- **结果**：峰值价值超过1600亿美元，是伯克希尔最大的单一持仓
- **后续**：2024年起大幅减持约三分之二
- **来源**：股东信、股东大会Q&A、CNBC采访
- **可信度**：⭐⭐⭐⭐⭐

### 5. 比亚迪（BYD）
- **时间**：2008年通过中美能源买入，约2.3亿美元
- **背景**：芒格推荐，称比亚迪创始人王传福是"托马斯·爱迪生和杰克·韦尔奇的合体"
- **决策逻辑**：芒格主导判断，看好电动车和电池技术的长期前景
- **结果**：持有至2022-2023年开始减持，投资收益超过30倍
- **来源**：股东大会、芒格采访
- **可信度**：⭐⭐⭐⭐

### 6. BNSF铁路（Burlington Northern Santa Fe）
- **时间**：2009年以约340亿美元全资收购
- **背景**：金融危机期间，铁路资产价格合理
- **决策逻辑**：铁路是美国经济的基础设施，不可能被复制（护城河 = 物理垄断）+ 环保优势（铁路比卡车高效4倍）
- **结果**：成为伯克希尔的核心资产之一
- **来源**：股东信
- **可信度**：⭐⭐⭐⭐⭐

## 二、重大失误与反思

### 1. Dexter Shoes（德克斯特鞋业）
- **时间**：1993年以4.33亿美元伯克希尔股票收购
- **失误**：用伯克希尔股票支付（等同于用一股无限增长的资产换了一家衰退企业），且Dexter因海外竞争丧失竞争优势
- **损失**：巴菲特称这是"我做过的最糟糕的交易"——不仅4.33亿归零，还因为用股票支付，那些股票今天的市值超过数十亿美元
- **教训**：（1）永远不要用股票收购（2）护城河可能比你想象的更脆弱
- **来源**：股东信（巴菲特多次在信中以此为反面教材）
- **可信度**：⭐⭐⭐⭐⭐

### 2. ConocoPhillips 时机错误
- **时间**：2008年油价高峰期间大举买入康菲石油
- **失误**：在油价接近峰值时买入，没有对未来油价做判断就行动
- **损失**：数十亿美元
- **反思**：巴菲特承认"没有预料到油价的暴跌"，犯了"非受迫性失误"
- **来源**：2008年股东信
- **可信度**：⭐⭐⭐⭐⭐

### 3. 错过Google和Amazon
- **背景**：巴菲特多次承认应该投资Google和Amazon
- **逻辑**：他对科技公司的偏见让他错过了消费品属性极强的Google广告业务和Amazon的电商护城河
- **反思**："我看得懂Google的广告模型，但还是没有行动，这是不可原谅的"
- **来源**：股东大会Q&A
- **可信度**：⭐⭐⭐⭐⭐

### 4. 不投航空股的食言
- **时间线**：2002年称航空股是"投资者的死亡陷阱" → 2016年买入四大航空公司 → 2020年新冠疫情中全部清仓
- **教训**：行业结构性问题（高固定成本、激烈竞争、工会压力）在疫情中暴露无遗
- **反思**："我是对的（航空是死亡陷阱），但中间犯了错（还是买了）"
- **来源**：2020年股东大会
- **可信度**：⭐⭐⭐⭐⭐

## 三、投资风格的标志性转变

### 从"烟屁股"到"优质企业"

| 维度 | 早期风格（格雷厄姆式） | 后期风格（芒格式） |
|------|----------------------|-------------------|
| 时期 | 1950s-1970s | 1972年至今 |
| 核心理念 | 以低于清算价值买入 | 以合理价格买入伟大企业 |
| 代表投资 | Sanborn Map、Dempster Mill | 可口可乐、苹果、See's Candies |
| 转变关键 | See's Candies（1972年）——第一次愿意为品质付溢价 |
| 转变原因 | 芒格的说服："以公平价格买入优秀企业，优于以优秀价格买入公平企业" |
| **巴菲特自评** | "我是85%格雷厄姆 + 15% 芒格"（后调整为更偏向芒格） |

### See's Candies：风格转变的里程碑
- **时间**：1972年，芒格和巴菲特以2500万美元收购
- **意义**：巴菲特第一次为一个"品牌护城河"支付远超账面价值的价格
- **结果**：See's此后产生了大量现金流，成为伯克希尔的"现金奶牛"
- **巴菲特评价**："是查理打开了我的眼界，让我从格雷厄姆的局限中跳出来"

## 四、组织与继任决策

### 接班人安排
- **现状**：Greg Abel被指定为CEO继任者（2021年股东大会确认）
- **投资负责人**：Todd Combs和Ted Weschler管理部分投资组合
- **芒格去世后**（2023年11月）：巴菲特的信和股东大会风格明显变化，更多回顾历史、更少展望未来
- **2025年**：巴菲特宣布年底卸任CEO，Greg Abel接任
- **来源**：股东大会、CNBC采访
- **可信度**：⭐⭐⭐⭐⭐

### 慈善决策：Giving Pledge
- **时间**：2010年与Bill Gates共同发起
- **内容**：承诺捐出99%以上的个人财富
- **累计捐赠**：截至2025年已捐赠超过560亿美元（主要是伯克希尔股票，捐给Bill & Melinda Gates Foundation等）
- **决策逻辑**："留给孩子的钱应该是'足够他们做任何事，但不够他们什么都不做'"
- **来源**：Giving Pledge官网、股东信
- **可信度**：⭐⭐⭐⭐⭐

## 五、发现的矛盾与不一致

1. **科技股原则 vs 买入苹果**：长期说不投科技股，2016年大举买入苹果。自评：苹果是消费品公司，不是科技公司。这个解释有道理但有事后合理化嫌疑。
2. **航空股态度反复**：2002年"死亡陷阱"→ 2016年买入 → 2020年清仓。承认自己"犯了老毛病"。
3. **衍生品立场 vs 实际操作**：批评衍生品是"大规模杀伤性金融武器"，但伯克希尔自己持有大量衍生品合约。自评：自己持有的是长期简单衍生品，与华尔街的复杂衍生品不同。
4. **被动投资推荐 vs 自己主动管理**：反复推荐普通人买指数基金，但自己做主动投资。这不矛盾但容易被误解。
5. **捐赠对象的争议**：2025年宣布不再向Bill & Melinda Gates Foundation捐款，引发关注。
