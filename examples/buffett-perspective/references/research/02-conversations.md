# 巴菲特调研 02：对话与深度采访

## 一、核心对话来源

### 1. 伯克希尔·哈撒韦年度股东大会Q&A
- 每年5月第一个周六，巴菲特+芒格坐镇回答股东提问长达5-6小时
- 来源：CNBC直播、YouTube存档、《杰出投资者文摘》transcript
- 可信度：⭐⭐⭐⭐⭐（一手，即兴回答最能揭示真实思维）
- 特点：幽默、坦率、经常用具体故事代替抽象理论

### 2. CNBC采访（Becky Quick系列）
- 多年多次深度采访
- 特点：巴菲特比较放松，会展开讲
- 来源：CNBC.com, Squawk Box

### 3. Charlie Rose访谈
- 长对话形式，覆盖人生哲学、投资、社会议题
- 特点：节奏慢，巴菲特更愿意谈抽象话题

### 4. PBS "Becoming Warren Buffett" 纪录片（2017）
- 巴菲特自己讲述人生故事
- 特别有价值：他谈到阅读习惯、第一段婚姻、与芒格的关系

## 二、被追问时的回答方式

### 经典模式：用故事代替说教
- 不直接回答"你怎么看X"，而是讲一个相关的故事
- 例：问"怎么看待当前市场"→ 讲"恐惧与贪婪"的故事
- 例：问"该不该买这只股票"→ 用棒球比喻（Ted Williams的击球区）
- 核心逻辑：好的回答不是给答案，是给一个能自己得出答案的框架

### 处理不知道的问题
- 直接说"我不知道"，不装懂
- "我不懂科技公司，所以我就不投" —— 用承认无知来保护自己
- 这种坦率是他的风格标签之一

## 三、即兴类比（最具特色的思维方式）

### 1. 棒球比喻
- "投资就像打棒球。Ted Williams只在他最有把握的区域挥棒。我也是。"
- 含义：能力圈 + 等待 + 只在最佳时机出手

### 2. 潮水退了
- "只有潮水退了，才知道谁在裸泳"
- 含义：牛市掩盖了风险，真正的质量在危机中才显现
- 首次出现：多次股东信引用

### 3. 老太太内裤
- "别人在舞会上穿得花枝招展，我们穿着老太太内裤"
- 含义：我们不跟风，只做自己认为对的事

### 4. 刺猬与狐狸
- "我是一个刺猬——只知道一件大事"
- 含义：专注 > 多元化（投资层面）

### 5. 20个打孔位
- "想象你有一张卡片，上面只有20个打孔位。每做一次投资就打一个孔。这会让你非常谨慎。"
- 含义：极端精选，质量 > 数量

## 四、改变立场的瞬间

### 最重要的立场转变：从不投科技股到买入苹果
- **早期**：反复说"我不懂科技，所以不投"
- **2016年**：开始买入苹果股票
- **他的解释**：苹果不是一家科技公司，是一家消费品公司——用护城河分析而不是科技分析
- **思维意义**：这展示了他不是固执的——当框架条件满足时，他愿意扩展能力圈
- **但注意**：他不认为自己"改变立场"了——他认为苹果在他的框架内，只是别人误解了他的框架

### 从"烟屁股"到"优质企业"
- **早期（1950-60年代）**：严格遵循格雷厄姆——买便宜的烂公司
- **受芒格影响后**：转向"以合理的价格买好公司，而不是以便宜的价格买烂公司"
- **他自己承认**：这是他投资生涯最重要的转变
- **经典表述**："查理打破了我捡烟屁股的习惯"

## 五、拒绝回答/回避的问题

### 政治立场
- 尽量不明确表态（尽管倾向于民主党）
- 回答方式："我不希望伯克希尔因为我的政治观点而失去客户"
- 但他在2016年支持希拉里，这是一个例外

### 接班人具体名单
- 多年来被反复追问谁会接班
- 早期拒绝透露，后来逐步给出线索（Ajit Jain, Greg Abel等）
- 2021年确认Greg Abel为CEO接班人

### 对其他投资者的直接批评
- 几乎从不公开批评同行
- 即使不同意，也会说"他们有自己的方法，我有我的"

---

## 六、与芒格的互动模式

### 典型模式
- 巴菲特长篇论述 → 芒格补充一句尖锐的话 → 巴菲特笑着同意
- 例：巴菲特讲5分钟的投资逻辑 → 芒格说"我没什么要补充的" → 全场笑
- 例：巴菲特讲完 → 芒格说"如果巴菲特没那么聪明，他可能会更安全一点"

### 思想互补
- 巴菲特：温和、用故事说服、不太尖锐
- 芒格：直率、毒舌、挑战一切
- 合在一起：刚柔并济的组合
